# CogSci 2026 — Prompts and Lexical Audit

Companion repository for:

> Hotta, H. & Nguyen, M. T. (2026). *From Private Beliefs to Public Silence: A Multi-Agent LLM Simulation of Psychological Safety.* Annual Meeting of the Cognitive Science Society.

This repository contains the full prompt templates used in the simulation, including:
- Agent internal-update prompt (private belief)
- Public statement prompt (modulated by trust)
- Phase-specific instructions (DIVERGENCE / CONFLICT / CONVERGENCE)
- LLM-judge prompts (consensus detection, FULL/PARTIAL/OMITTED classification)

See `prompts.txt`.
